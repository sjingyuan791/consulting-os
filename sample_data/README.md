# サンプルデータ テンプレート一覧

## フォルダ構成

```
sample_data/
├── 01_財務データ/          ← 財務分析用データ
│   ├── 決算書.csv
│   ├── 借入一覧.csv
│   ├── 月次推移.csv
│   ├── 資金繰り予定表.csv
│   ├── 固定資産台帳.csv
│   └── 予算実績対比.csv
│
└── 02_内部環境データ/      ← 内部環境分析用データ
    ├── 従業員一覧.csv
    ├── 得意先一覧.csv
    ├── 仕入先一覧.csv
    ├── 商品・在庫マスタ.csv
    ├── 売上明細.csv
    └── 部門別損益.csv
```

---

## 01_財務データ（Financial Data）

| ファイル | 用途 | 主な分析 |
|----------|------|----------|
| **決算書.csv** | BS/PL全勘定科目（90項目） | 財務分析、CF計算、EBITDA |
| **借入一覧.csv** | 借入先・返済条件・残高 | 返済能力、リスケ検討 |
| **月次推移.csv** | 月次売上・経費・残高 | 季節変動、トレンド |
| **資金繰り予定表.csv** | 入出金予定 | 資金ショート予測 |
| **固定資産台帳.csv** | 固定資産・減価償却 | 担保評価、減損 |
| **予算実績対比.csv** | 予実管理 | 予算達成率分析 |

---

## 02_内部環境データ（Internal Environment）

| ファイル | 用途 | 主な分析 |
|----------|------|----------|
| **従業員一覧.csv** | 人員・給与・資格 | 人件費分析、組織診断 |
| **得意先一覧.csv** | 取引先・売掛金・与信 | 与信管理、顧客分析 |
| **仕入先一覧.csv** | 仕入先・買掛金 | 調達コスト、リスク |
| **商品・在庫マスタ.csv** | 商品・在庫・原価 | ABC分析、滞留在庫 |
| **売上明細.csv** | 日次売上明細 | 売上分析、粗利管理 |
| **部門別損益.csv** | 部門・セグメント別収益 | セグメント分析 |

---

## 使い方

1. 該当するテンプレートをダウンロード
2. ExcelまたはCSV編集ソフトで開く
3. 実際のデータを入力
4. アプリにアップロード

---

## 注意事項

- 金額単位は**万円**（または円、テンプレートによる）
- 日付形式は **YYYY/MM/DD**
- 空白セルは0または未入力として処理されます
